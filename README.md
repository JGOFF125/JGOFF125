## About

I am a **Software Development student at SAIT** and an aspiring software engineer based in **Calgary, Alberta, Canada**.

I build software with an emphasis on maintainability, user experience, security and real-world usefulness. My current work includes mobile application development, relational databases, software modelling, frontend development, applied artificial intelligence and collaborative Git workflows.

My primary technologies include **TypeScript, JavaScript, Python, C#, Java, React Native, Expo, SQL, Oracle APEX, Git and GitHub**.

I enjoy working across the complete software-development lifecycle: understanding the problem, designing the system, implementing the product, debugging technical issues, testing the experience and continuously improving the result.

I approach development with a product-engineering mindset. I want to understand the user, remove unnecessary complexity and create software that can evolve beyond its initial version.

My long-term goal is to become a highly capable software engineer who can build scalable technology products, lead engineering projects and create companies that solve meaningful problems.
